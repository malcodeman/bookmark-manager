Bookmark manager for power users.

## Getting started

```
git clone https://github.com/malcodeman/bookmark-manager
cd bookmark-manager
npm install
npm run dev
```

## Testing

```
"test": "jest --watch",
"test:ci": "jest --ci",
"cypress:open": "cypress open",
"cypress:ci": "cypress run --headless"
```
